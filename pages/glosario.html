Glosario de terminos

Terminos tecnicos propios de ClickUp y gestion de proyectos que aparecen en este manual.

Workspace

Nivel mas alto de organizacion en ClickUp. Contiene todos los Spaces, miembros y configuraciones.

Space

Unidad de organizacion dentro del Workspace. Equivale a un departamento o proyecto grande.

Folder

Agrupacion opcional dentro de un Space para organizar Lists relacionadas.

List

Contenedor de tareas con estados configurables y multiples vistas disponibles.

Task (Tarea)

Unidad basica de trabajo. Tiene nombre, descripcion, asignados, fechas, prioridad y mas.

Subtask

Tarea hija dentro de una tarea principal. Permite desglosar trabajo complejo en pasos pequenos.

Assignee

Persona responsable de completar la tarea. Una tarea puede tener multiples assignees.

Due Date

Fecha y hora limite para completar una tarea. Diferente de Start Date (inicio).

Priority

Nivel de urgencia: Urgente (rojo), Alta (amarillo), Normal (azul), Sin prioridad (gris).

Status

Estado actual en el flujo de trabajo: To Do → In Progress → Done. Personalizable por List.

FAB

Floating Action Button. El boton "+" morado para crear tareas rapidamente desde cualquier vista.

Tag

Etiqueta personalizable para categorizar tareas transversalmente en el workspace.

Inbox

Centro de notificaciones personal: tareas asignadas, menciones y recordatorios.

CTA

Call to Action. Elemento que invita al usuario a una accion especifica (ej: boton "Create task").
